# 智能背单词应用

这是一个单文件的智能背单词应用，支持单词管理、学习、个人计划和基于遗忘曲线的复习推荐。

## 功能特性

- **单词管理**：支持添加自定义单词，包含单词、释义和例句
- **学习功能**：随机展示单词，记录学习情况
- **个人计划**：设置每日学习目标和学习时间，查看学习进度
- **复习推荐**：基于艾宾浩斯遗忘曲线，推荐最佳复习时间

## 技术实现

- **前端技术**：HTML5, CSS3, JavaScript
- **数据存储**：使用浏览器的localStorage
- **响应式设计**：适配不同屏幕尺寸
- **跨平台兼容**：支持在多种设备上打开

## 如何使用

### 方法一：直接在浏览器中打开

1. 下载 `index.html` 文件到本地
2. 双击该文件，在浏览器中打开
3. 开始使用背单词功能

### 方法二：在GitHub上托管

1. 创建一个新的GitHub仓库
2. 将 `index.html` 文件上传到仓库
3. 启用GitHub Pages：
   - 进入仓库设置
   - 找到 "GitHub Pages" 选项
   - 选择 "main" 分支作为源
   - 点击 "Save"
4. 等待几分钟，GitHub会生成一个访问链接
5. 通过该链接在任何设备上访问应用

### 方法三：部署到其他静态网站托管服务

可以将 `index.html` 文件部署到以下服务：
- Netlify
- Vercel
- Firebase Hosting
- AWS S3 + CloudFront

## 功能说明

### 学习新单词
- 点击 "学习新单词" 按钮
- 系统会随机展示单词和释义
- 点击 "认识" 或 "不认识" 按钮记录学习情况
- 系统会根据你的回答安排后续的复习时间

### 上传单词
- 点击 "上传单词" 按钮
- 填写单词、释义和例句（可选）
- 点击 "添加单词" 按钮保存
- 下方会显示已添加的单词列表

### 个人计划
- 点击 "个人计划" 按钮
- 设置每日学习目标（单词数）和学习时间（分钟）
- 点击 "保存计划" 按钮确认
- 页面会显示今日学习进度和完成情况

### 复习
- 点击 "复习" 按钮
- 系统会根据艾宾浩斯遗忘曲线算法，显示今日需要复习的单词
- 复习完成后，系统会重新安排下一次复习时间

## 数据存储

- 所有数据存储在浏览器的localStorage中
- 数据仅存储在本地，不会上传到服务器
- 清除浏览器缓存会导致数据丢失

## 浏览器兼容性

- Chrome 60+
- Firefox 55+
- Safari 10+
- Edge 79+

## 移动设备支持

- 支持在手机浏览器中打开
- 响应式设计，适配不同屏幕尺寸
- 支持触摸操作

## 注意事项

- 首次使用时，系统会自动初始化默认单词数据
- 建议定期备份localStorage中的数据
- 如果在微信中打开遇到问题，可以尝试使用其他浏览器
